# New Proyect - Landing Page

Landing page profesional para **New Proyect**, empresa de Ingeniería y Mantenimiento Industrial Integral.

## Características

- Diseño moderno, minimalista y elegante
- Responsive (Mobile First)
- Galería con slider e imágenes de proyectos
- Lightbox para ver imágenes en grande
- Botón flotante de WhatsApp
- Animaciones con AOS (Animate On Scroll)
- Navbar con efecto glassmorphism

## Stack

- HTML5 semántico
- Tailwind CSS (CDN)
- Vanilla JavaScript (ES6+)
- Swiper.js (slider)
- GLightbox (galería)
- Font Awesome

## Ejecución local

Abre el archivo `index.html` en el navegador o usa un servidor local:

```bash
python3 -m http.server 8080
```

Luego visita http://localhost:8080

## Estructura

```
├── index.html      # Landing page principal
├── docs/           # Documentación y especificaciones
│   ├── specs.md           # Especificaciones del proyecto
│   ├── specs-servicios.md # Especificaciones sección servicios
│   └── INFO EMPRESA/      # Información de la empresa
└── assets/         # Imágenes y recursos (fotos de proyectos, etc.)
    ├── logos/      # Logos de la empresa
    └── servicios/  # Fotos por servicio (END, Integridad Mecánica, etc.)
```
