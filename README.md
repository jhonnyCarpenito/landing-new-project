# New Proyect - Landing Page

Landing page profesional para **New Proyect**, empresa de Ingeniería y Mantenimiento Industrial Integral.

## Características

- Diseño moderno, minimalista y elegante
- Responsive (Mobile First)
- Galería con slider e imágenes de proyectos
- Lightbox para ver imágenes en grande
- Botón flotante de WhatsApp
- Animaciones con AOS (Animate On Scroll)
- Navbar con efecto glassmorphism

## Stack

- HTML5 semántico
- Tailwind CSS (CDN)
- Vanilla JavaScript (ES6+)
- Swiper.js (slider)
- GLightbox (galería)
- Font Awesome

## Ejecución local

Abre el archivo `index.html` en el navegador o usa un servidor local:

```bash
python3 -m http.server 8080
```

Luego visita http://localhost:8080
